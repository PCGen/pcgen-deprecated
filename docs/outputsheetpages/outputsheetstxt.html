csheet_plain.TXT

Character Output Sheets: .rpgrp Based Versions

.rpgrp is an xml based format for RPTools combat initiative tracking program InitTool. The two sheets provided output to InitTool's group file format. Once exported to this format InitTool can open the file with the group tool or the characters can be directly added to the initiative window using the group button. The graphic assigned by PCGen as the character portrait is used as the token for the character in InitTool.

You must choose "File > Export > ...to Text" to select these sheets for output.

csheet_InitTool.rpgrp

  • Outputs a single character to InitTools group file format.

psheet_InitTool.rpgrp

  • Outputs the entire party to InitTools group file format. You must check "Export Entire Party" to select this sheet.
