Main Tab: Feats

This is the Feats tab. Here the user will choose which feats and other abilities, loaded from the sources, to add to the character.

As of version 5.12 the Feats Tab has sub-tabs to support multiple feat-like abilities. By default the Feats sub-tab will always be first. Additional sub-tabs other other categories of abilities may be present. Currently only the RSRD's Divine Salient Abilities and the MSRD's Mutations are using this new feature. Expect to see much more development in future versions, this is the planned place for displaying and choosing Class Features, Racial Traits, Special Qualities and Special Attacks as well as other, more exotic features used in other datasets.

The number in the upper-left contains the list of Feats (or Abilities) to be chosen.

Next to the word "Available" is a selection box that lets the user pick which "view" to use. It is this view, which will be used to determine the structure of the table below it. The same is true for the "Selected" feats, which list the feats already selected for the PC.

By default the feats for the PC'selected as optional feats are in black text in the Selected table. Feats and Abilities gained automatically (due to a race or class property) are displayed in dark-yellow text. Feats and Abilities gained virtually (typically this means there's some restriction on them) are displayed in magenta text. Virtual or Automatic Feats and Abilities cannot be removed or changed the selections of Virtual or Futomatic Feats and Abilities. By clicking on the Filter icons in the toolbar (the icons with the magnifying glass.)

To add an item, double-clicking on it, right clicking on it or clicking on it and the on the right-arrow button. Similarly to remove feats, or alter the selections of optional feats which have choices, by double-clicking on it in the "Selected" table, or by right-clicking on it there, or by selecting it and then clicking on the left-arrow button above the table. As elsewhere in PCGen anything that is not qualified for is in red italics.

The info-pane in the bottom-left will provide details about any feat clicked upon - including any pre-requisites and its source.
