Domain Files

The Domain files are used to specify domain pertinent information The first field is the Domain Name, this is a required field and does not have a tag.

Tag Name: FEAT:x,x

Variables Used (x): Text (Feat Name)

What it does:

  • This is a comma (,) delimited list of feats granted to a Race.
  • The hyphen (-) is used only in the OUTPUTNAME of feats and should NOT be used in the feat list for this tag. (See example below.)

Example:

FEAT:Blind Fight,Alertness

The "Blind-Fight" & "Alertness" bonus feats are granted to clerics with this domain.
