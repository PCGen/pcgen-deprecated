Feat Editor: Weapon Tab

The Weapons Tab is used to add a Granted Weapon Proficiencies for the Feat being created.

The two Granted Weapon Proficiencies windows, Available and Selected are used to create a list of Granted Weapon Proficiencies.

  • The Add and Remove buttons will move the highlighted Granted Weapons between the 2 windows, as will double clicking on a Granted Weapons.

The Cancel and Save buttons, which appear on every tab, are used to either cancel the Feat creation or save it to the customFeats.lst file.
